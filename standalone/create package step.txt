Create  Package Step

1)  Create Ezybaas-package Folder

2)  Create setup.py , mainifest.in , readme.rst and Licence File

3)  Add project Folder in Ezybaas-package Folder
4)  Add manage.py file in ezybaas-package folder
5)  Go to ezybaas-package or django-ezybaas in terminal
4)  Run "python setup.py sdist "
